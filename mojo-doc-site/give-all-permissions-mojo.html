gosdk-wrapper:give-all-permissions

Full name:

com.igormaznitsa:gosdk-wrapper-maven-plugin:1.0.2:give-all-permissions

Description:

Try give all permissions to files defined through file sets. This mojo walks through all files and trying to provide all permissions to work with them and remove flags which can make the files non-accessible.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.2.
  • Binds by default to the lifecycle phase: pre-clean.
